# 👶🎉 Baby Maddula’s Gender Reveal Quest

Welcome to **Baby Maddula’s Gender Reveal Quest**, a joyful and interactive mini-game built to celebrate the exciting moment of discovering Baby Maddula’s gender!

This project is a fun, browser-based game created using pure HTML, CSS, and JavaScript — no frameworks or backend required.

---

## 🌟 Features

- 🍼 **3-Level Interactive Game**
- 🎵 Lullaby quiz, image hotspot challenge & key guessing
- ⏳ Countdown timer for added excitement
- 🎈 Animated balloons and themed reveal screen
- 🔊 Background music to enhance the experience
- 📱 Mobile-friendly layout (with slight modifications)
